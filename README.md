# n-alkanes_Verdeospesoa

This repository contains an R workflow for analysing n-alkanes from the Verdeospesoa peat bog
This repository contains the directories "code" and "data" to reproduce the analyses, figures and tables of the paper: Climate, vegetation and soil moisture dynamics across the Pleistocene-Holocene transition in Northern Iberia: evidence from the Verdeospesoa peat bog. 
Data and code in this repository can be reproduced with only an R environment in RStudio. The necessary package dependencies are documented in each .R file.
